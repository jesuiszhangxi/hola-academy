# 写在前面

## 目标和范围

本教程的学习目标是实现西班牙语的日常使用：生活中的对话，简单的日常阅读和写作（例如生活中的短信、邮件）。为了达到这个目的，本教程覆盖了日常生活中最主要场景（现在，过去，将来）的基本西班牙语语法，但不追求高端和复杂的语法。

本教程学习的是拉美西语（Español latino），而非西班牙的西班牙语（Español de España）。二者的关系有点像美式英语和英式英语。以下两个原因导致前者具有更高的“性价比”：

1. Español latino适用范围更广。全球90%的西语母语者讲的是Español latino。全球第一大西语国家是墨西哥，第二大西语国家是美国（拥有6500万西语人口！），讲的都是Español latino。所以Español latino是比Español de España更有国际影响力的西班牙语。

2. Español latino更简单。它在发音和语法进行了一些简化。例如，Español latino抛弃了第二人称复数vosotros（你们），而一律使用敬语ustedes（您诸位，语法上等价于第三人称复数），所以动词变位从6个减少到了5个，降低了记忆负担。

本教程的基本原则是实用——“**非必要不学习**”。如果一个语言场景可由简单方案解决，则不学习更复杂的方案。比如，基于ir+a+inf.结构的“近将来时”可描述大部分涉及到将来的场景，那么我们就只学习这种近将来时（一个小时即可完成），而不去学习基于动词变位的将来时。

基于这一原则，在时态方面，我们在本教程里将学习以下几个时态来覆盖“现在-过去-将来”这个基本的日常生活时间轴：

1. 陈述式现在时

2. 近将来时

3. 未完成过去时

4. 简单过去时

5. 现在进行时

## 本教程适合谁？

* 已学习过英语的中文母语者。注意：本教程并不是给English speaker的西班牙语教程。本教程是**面向具有一定英语水平的中文母语者的西班牙语教程**。那么本教程对学习者的英语水平的要求有多高呢？差不多高中英语水平即可。具体来说，你需要理解英语的基本语法结构、常用时态、主宾格、单复数等知识点。

* 喜欢逻辑思维和基于规则的语言学习，而非基于感性记忆（死记硬背）的语言学习方式。由于其语言的“纯粹性”，加上统一管理者（RAE）在过去300年的维护工作，西班牙语是一个逻辑性和规则性非常强的语言——其大部分语法都是基于规则和逻辑来展开的。如果能掌握这些规则和逻辑，学起来会事半功倍。当然，前提是你喜欢这种基于逻辑和规则的语言学习方式。

## 本教程不适合谁？

* 如果你完全不会英语，或者说西班牙语是你学习的第一门外语，那么你可能需要一本类似《现代西班牙语》那样的“第一外语启蒙”教材，帮你从零开始引入各种概念。

* 喜欢基于感性记忆和自然经验的学习模式，而不喜欢基于逻辑和规则的学习者。本教程的基本思路是规则化一切能规则化的地方，**尽最大的可能基于规则去学习语言**。但这种学习方式并不适合所有人。

* 学习目的是高端使用场景的学习者。比如，如果你希望进行复杂文本撰写（论文、商业合同等）、或者文学写作，那本教程不足以你支撑你从事此类语言活动。但是你可以把本教程当作第一个阶段的学习，完成之后再进入高层次学习。

## 常规西语教学的问题是什么？

这个问题的答案也同时解释了我为什么要编写当前这个教程。因为我觉得中国常规的西语教学体系（至少它所使用的大部分教材所反映出的思路）是错的。

中国常规的西语教学在教材上采用的是“外语启蒙教学”模式。例如最有代表性的外研社的《现代西班牙语》（曹燕生/刘建著）教授西班牙语的方式跟中国中小学教授英语的方式几乎一模一样，而完全没有考虑到中国的西语学习者在学习西班牙语时都已经学习过了英语这一事实。事实上，中国很少有人在不会英语的情况下直接学习西班牙语。绝大多数中国人在中学阶段已经通过学习英语建立了印欧语系的语言框架，对句子结构、从句结构、时态、格、数已经非常熟悉，对动词变位（am/are/is）也有所接触（只是在英语中并不普遍）。中国人学习西班牙语所遇到的唯一全新概念，可能只有阴阳性。

在这种情况下，完全不需要像初中（有些地方是小学）作为“第一外语启蒙”的英语课程那样去学西班牙语，而完全可以用一种更高效的方式——**基于英语所建立的印欧语言框架来学习**——但我几乎没有看到任何一个教学体系是这么做的。至于Duolingo这样的语言学习app，采用的则是幼儿园式的“纯感性”教学，当作打发时间的游戏来“寓教于乐”是可以的，但如果仅依赖于它学习西班牙语，那效率就太低了——可能永远也学不会语法。

## 为什么要通过英语学习西班牙语？

简单来说，完全基于中文学习西班牙语的学习曲线非常陡峭。**但是如果以英语为桥梁，学习曲线则平缓得多**。而且这是一座现成的、免费的桥梁——因为它早已存在于你的认知体系中。

我们知道，大多数外语学习的四大难点是：语法、单词、听力、口语。其中后三项几乎没有捷径，只能靠练。而语法则各语言难易不同。西班牙语语法对中文母语者来说最大的障碍在于它跟中文分属不同的语系：西班牙语属于**屈折语系**，而中文属于**分析语系**。西班牙语大量的语法是基于单词本身的形态变化——这是中文这种分析语完全不存在的语法现象。而英语作为一种既有分析语语法又有屈折语语法的综合语，已经帮中文母语者建立了基本的屈折语概念（比如基于ing/ed后缀的时态，动词第三人称单数加s，等等）。英语的各种语法结构虽然不能跟西班牙语精确对应，但是其对应程度远远高于中文跟西班牙语的对应程度。这就是为什么英语是中文母语者学习西班牙语最好的桥梁。

## 本教程没有包含什么？

本教程只包含覆盖基本的日常“现在-过去-将来”时间轴的基本时态，除此之外的“高级”时态均不包括。关于虚拟语气，本教程仅覆盖几个基本句式来满足日常基本场景的需求，而不对虚拟语气进行系统化学习。

以下三项是语言学习的必要组成部分，但是本教程没有覆盖，需要单独学习和练习：

+ 单词学习。本教程除教学所涉及到单词的之外，并不包含单词表。请把“背单词”作为一项独立的学习项目进行。

+ 听力

+ 口语

Note：虽然Duolingo那种“纯感性学习”效率较低，但是如果作为本教程的补充——用来学习单词、听力、口语，那还是非常有用的。
